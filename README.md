# ⚙ Installation

```bash
git clone https://github.com/sandip-kumar-jha/Ai-cover-letter-generator.git
```

```bash
cd Ai-cover-letter-generator
```

```bash
npm install
```

Create a `.env` file:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

Run the project:

```bash
npm run dev

---

# 📖 How It Works

1. Enter Candidate Name.
2. Enter Job Role.
3. Enter Target Company.
4. Enter Key Skills.
5. Upload Resume (Optional).
6. Click **Generate Cover Letter**.
7. Gemini AI generates a professional ATS-friendly cover letter.
8. Copy the generated cover letter instantly.

---

# 🔐 Environment Variables

Never expose your API Key.

Create:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

Make sure `.env` is inside `.gitignore`.
